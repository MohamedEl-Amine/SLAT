## Quick Reference

### When Can I Use the Camera?

The camera is **only available during working hours**:
- **Morning Window**: Arrival time (check with admin)
- **Afternoon Window**: Departure time (check with admin)

Outside these hours, the camera is **automatically disabled** for security and privacy.

### How to Activate the Camera

1. **Check the screen** - You'll see instructions if the camera is available
2. **Press SPACE or ENTER** on the keyboard
3. **Camera will activate** and show a countdown timer
4. **Present your QR code** or **look at the camera** (depending on mode)

### Troubleshooting

#### Camera Won't Activate
1. Check if you're within working hours
2. Make sure you're in QR or Face mode (not Card ID mode)
3. Try pressing SPACE or ENTER again
4. If error persists, switch to Card ID mode (press TAB)

#### Timer Keeps Expiring Too Fast
- The 2-minute timer is working correctly
- After each scan, it automatically resets to 2:00
- If camera closes, just press SPACE to reactivate

#### Camera Activated But Not Scanning
- Make sure to present QR code clearly
- For face mode, look directly at camera
- Wait for recognition (there's a 3-second cooldown between scans)
- Check the countdown timer - if it shows red, camera may close soon
